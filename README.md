# AI-STUDY--BUDDY# 📚 Study Buddy

## About
Study Buddy is a simple web application built using Python and Flask. It provides a clean interface where users can ask study-related questions and interact with the application.

## Features
- Simple and user-friendly interface
- Built with Python and Flask
- Beginner-friendly project
- Easy to customize and extend

## Tech Stack
- Python
- Flask
- HTML
- CSS

## How to Run

```bash
pip install flask
python app.py
```

Then open:

```
http://127.0.0.1:5000
```
